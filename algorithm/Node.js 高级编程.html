# 全局对象

# Nodejs 全局对象

  • 与浏览器平台的window不完全相同
  • Nodejs全局对象上挂载许多属性

# 全局对象是JavaScript中的特殊对象

# Nodejs中全局对象是global

# Global的根本作用就是作为宿主

# 全局对象可以看做是全局变量的宿主

# Nodejs 常见全局变量

  • filename：返回正在执行脚本文件的绝对路径
  • dirname：返回正在执行脚本所在目录
  • timer类函数：执行顺序与事件循环间的关系
  • process：提供与当前进程互动的接口
  • require：实现模块的加载
  • module、exports：处理模块的导出
